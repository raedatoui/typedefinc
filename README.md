# Typedef
To learn more about Typedef, visit [typedefinc.com](https://typedefinc.com).

## Learn More
A Next.js project featuring fragment shader art and 3D scenes using React Three Fiber.

## Getting Started

First, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
